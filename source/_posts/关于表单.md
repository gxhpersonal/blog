### 表单
> 关于input表单的一些样式和JS问题：
input的placeholder属性字体颜色：
```css
        input::-webkit-input-placeholder {
            color: #aaa;
        }
        input:-moz-placeholder {
            color: #aaa;
        }
        input::-moz-placeholder {
            color: #aaa;
        }
        input:-ms-input-placeholder {
            color: #aaa;
        }
```
input的选中状态：
```css
       input:focus{ border:1px solid red }
```
