如何将本地文件上传到Github上？

1.在目录中创建新的 Git 仓库

进入到你本地项目的根目录下，执行 git init 命令，就可以创建一个 Git 仓库了

2、执行 git add . 命令，将项目的所有文件提交到暂存区

3、执行git commit -m “注释语句” 命令，将暂存区内容添加到仓库中

4、在 Github 新建一个 Repository，获取所创建的仓库的HTTPS地址

5、将本地的仓库关联到Github上，上传代码到github远程仓库

bingo！
